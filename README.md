# Nitori Gallery Downloader

Nitori Gallery の画像および動画を自動で取得・保存するためのダウンローダーです。
Playwright を使用してブラウザ操作を自動化し、認証情報の取得から動画のストリーミング解析までをシームレスに行います。

## 主要機能
- **半自動認証**: 初回実行時にブラウザで一度ログインするだけで、セッション（LocalStorage/Cookie）を自動保存・再利用します。
- **画像一括ダウンロード**: アルバムごとにフォルダを自動作成し、高画質な画像を保存。
- **動画ストリーミング解析**: HLS（.m3u8）リクエストをブラウザレベルでスニッフィングし、署名付きURLをキャプチャして `yt-dlp` でダウンロードします。
- **重複スキップ**: 既存のファイル（画像・動画）を検知し、未取得のコンテンツのみを効率的に処理。
- **詳細ロギング**: `downloader.log` に実行ログを記録し、トラブルシューティングを容易にします。

## セットアップ

### 1. 依存パッケージのインストール
```bash
pip install -r requirements.txt
```

### 2. Playwright ブラウザのインストール
```bash
python -m playwright install chromium
```

### 3. 環境設定
`.env.sample` を参考に `.env` ファイルを作成し、必要に応じて設定を調整してください。

## 使用方法
以下のコマンドでプログラムを実行します。
```bash
python nitori_downloader.py
```

### 初回実行時
1. 自動的にブラウザが立ち上がります。
2. ログイン画面が表示されるので、手動でログインを完了させてください。
3. トップページが表示され、コンソールに「認証情報を取得しました！」と表示されたらブラウザが閉じ、自動ダウンロードが開始されます。

### 2回目以降
保存された `session.json` を使用するため、ログイン操作なしで実行可能です。セッションが切れた場合は再度ブラウザが立ち上がります。

## ディレクトリ構成
- `nitori_downloader.py`: メインスクリプト
- `session.json`: 保存された認証セッション（自動生成）
- `cookies.txt`: yt-dlp 用のクッキーファイル（自動生成）
- `downloader.log`: 実行ログ（自動生成）
- `downloads/`: 画像・動画の保存先

## 注意事項
- **セキュリティ**: `session.json` および `cookies.txt` には個人セッションが含まれます。公共のリポジトリ等にアップロードしないでください。
- **規約遵守**: 取得したコンテンツの利用は個人利用の範囲に留め、サイトの利用規約を遵守してください。
- **リソース**: 動画のダウンロードには FFmpeg 等がシステムにインストールされている必要がある場合があります（yt-dlp の要件）。
