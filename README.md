# 👑 Nitori Gallery 真・全自動ダウンローダー 👑

Nitori Gallery の画像を、API なりすまし技術を使って根こそぎ救出するための最強ツールだよ！✨💎🚀

## 🌟 これができること
- **全自動全ページ巡回**: 2ページ目以降も自動でめくって、全アルバムを捕捉！🎯
- **高画質画像保存**: アルバム名をフォルダにして、連番でスマートに保存。
- **安心・安全**: 既に持ってる画像はスキップする賢い機能付き。

## 🛠️ クイックスタート（3ステップ！）

### 1. 魔法の源泉（HAR）を準備！
ブラウザでギャラリーページを開いたまま、以下の手順でファイルを保存してね：
1. `F12` キーを押して「Network（ネットワーク）」タブを開く。
2. そのままギャラリーを表示したあと、Networkタブ内の「↓（エクスポート）」ボタンを押す。
3. ファイル名を **`s-nitori.com.har`** にして、このプロジェクトフォルダに保存！
   - ※ これが「ログインの証」になるよ！

### 2. 下準備（初回のみ）
```bash
pip install -r requirements.txt
```

### 3. 救出ミッション開始！！
```bash
python nitori_downloader.py
```
あとは、お宝がザックザク溜まっていくのを眺めるだけ！💕

## 📁 フォルダの見方
- `nitori_downloader.py`: 本体のスクリプト
- `s-nitori.com.har`: これがないと動かないよ！命のファイル！
- `downloads/`: 救出したお宝（画像）がここに眠ってるよ📁✨
- `.env`: 保存先を変えたいときはここをいじってね。

## ⚠️ 大事な約束（セキュリティ）
- **公開禁止**: `s-nitori.com.har` や `session.json` にはご主人様の秘密が入ってるよ。絶対にネットに流しちゃダメ！！🙅‍♀️
- **期限切れ**: 「エラーが出た！」って時は、だいたい Cookie の期限切れだよ。手順1に戻って新しい HAR を録り直してね。
