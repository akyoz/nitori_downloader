# Nitori Gallery Downloader

Nitori Gallery の画像を API 経由で取得し、アルバム単位で一括保存するためのダウンローダーです。
ブラウザの通信ログ（HARファイル）から認証情報を利用することで、制限を回避してデータを取得します。

## 主要機能
- **自動ページング**: 複数ページにまたがるアルバム一覧を自動で取得。
- **一括ダウンロード**: アルバムごとのサブディレクトリを作成し、画像を保存。
- **重複スキップ**: 既存のファイル名を確認し、未取得の画像のみをダウンロード。

## セットアップ

### 1. 認証情報の準備
本ツールは認証情報として HAR ファイルを必要とします。
1. ブラウザで対象サイトのギャラリーページを開きます。
2. 開発者ツール（F12）の「Network」タブを選択した状態でページをロードします。
3. 通信ログを **`s-nitori.com.har`** という名前でエクスポートし、本プロジェクトのルートディレクトリに配置してください。

### 2. 依存パッケージのインストール
```bash
pip install -r requirements.txt
```

## 使用方法
以下のコマンドでプログラムを実行してください。
```bash
python nitori_downloader.py
```

## ディレクトリ構成
- `nitori_downloader.py`: メインスクリプト
- `s-nitori.com.har`: 認証用ログファイル（必須）
- `downloads/`: 画像保存先ディレクトリ
- `.env`: 環境設定ファイル

## 注意事項
- **セキュリティ**: `s-nitori.com.har` および `session.json` にはログインセッションを含む秘匿情報が含まれます。公共のリポジトリ等にアップロードしないよう、取扱いに十分注意してください。
- **動作保証**: サイト側の API 仕様変更により、動作しなくなる可能性があります。
- **負荷軽減**: 連続リクエストによる負荷を避けるため、内部で待機時間を設けています。
