## 🛠️ プロジェクト概要
このツールは、**s-nitori.com（ファンサイト）から画像をゴッソリ自動保存**するための、有能美少女なPythonスクリプトだよ！✨
普通のスクリプトとは一味違う、セキュリティと利便性を両立した「ギャルメイド仕様」になってるのがポイント！

## ✨ 搭載済み「ツヨツヨ」機能一覧

### 1. 🎀 ログインGUI & 自動保存
- **概要**: `.env` ファイルに情報を書かなくても、初回起動時に可愛いピンクの入力画面が爆誕するよ！
- **仕組み**: 標準の `tkinter` を使ってるから、追加ライブラリ不要で動くのがエラいっ。
- **保存**: 「保存する」にチェックを入れれば、`.env` を勝手に作って次からラクできるようにしちゃう！

### 2. 🎫 セッション管理（トークン永続化）
- **概要**: 毎回パスワードを入れるのはダルいから、ログインの「通行証（IDトークン）」を `session.json` に保存するようにしたよ！
- **有効期限チェック**: 起動時に「このトークンまだ使えるかな？🔍」って実際にサーバーに確認しにいく賢い設計。
- **メリット**: パスワードを平文で持ち歩かなくて済むからセキュアだし、有効な間は爆速でスタートできるよ🚀

### 3. 🔍 高度な Nuxt.js データ解析
- **概要**: サイトの裏側にある `window.__NUXT__` オブジェクトを直接狙い撃ち！
- **ID抽出**: HTMLタグをパースするんじゃなくて、ソース全体のテキストから正規表現でアルバムIDを炙り出す「力技」を搭載。
- **エスケープ対応**: `/` が `\u002F` になってても、アンティの眼（正規表現）からは逃げられないよ！👁️✨

### 4. 📁 スマート・フォルダ管理
- **概要**: 保存されるフォルダ名を、ただのIDじゃなくて「ID - アルバムタイトル」にするようにしたよ。
- **タイトル取得**: アルバムページの中身から、タイトルの文字列を自動で引っこ抜くよ。
- **安全設計**: フォルダ名に使えない `/` などの文字は、自動で `_` に置換してエラーを防ぐよ💪

## 🛡️ セキュリティへのこだわり
- **.gitignore 完備**: `.env` や `session.json` をうっかり GitHub に上げちゃわないように、しっかり見張ってるよ！
- **トークン方式**: 生のパスワードを使い続けるんじゃなくて、期限付きのトークンでアクセスする、プロっぽい仕様だよ✨

## 🚀 今後の課題（ご主人様、要チェック！😭）
最新のテストで「1ページ目からアルバムが見つからない」ってなっちゃったのは、以下の理由が考えられるよ：
1. **プランの壁**: 有効な有料サブスクリプションがないと、ギャラリーが表示されないのかも...？
2. **APIの構造変化**: サイト側が、HTMLソースの中じゃなくて、後からAPIを叩いてデータを取る方式に完全移行しちゃった可能性。
3. **データ配置の変更**: `window.__NUXT__` の場所が、私の想定している場所と微妙にズレちゃってるのかも。
